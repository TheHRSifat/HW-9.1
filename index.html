3.Compact

The compact attribute is a boolean attribute. When present, it specifies that the list should render smaller than normal, by reducing the space between lines and the indentation of the list.

5.Autofocus

The autofocus attribute is a boolean attribute. When present, it specifies that an input element should automatically get focus when the page loads.

6.Autoplay

The autoplay attribute is a boolean attribute. When present, the video will automatically start playing as soon as it can do so without stopping.

7.Contenteditable

Example:

The contenteditable attribute specifies whether the content of an element is editable or not

8.Controls

Example:

The controls attribute is a boolean attribute. When present, it specifies that audio/video controls should be displayed. Controls should include:

  • Play
  • Pause
  • Seeking
  • Volume
  • Fullscreen toggle (for video only)
  • Captions/Subtitles (for video only, when available)
  • Track (for video only, when available)

9.Default

Example:

The default attribute is a boolean attribute. When present, it specifies that the track is to be enabled if the user's preferences do not indicate that another track would be more appropriate.

10.formnovalidate

Example:

E-mail:

The formnovalidate attribute is a boolean attribute. When present, it specifies that the input element should not be validated when submitted. The formnovalidate attribute overrides the novalidate attribute of the form element.

11.frameborder

Example:

The frameborder attribute is a boolean attribute. The frame tag is not supported in HTML5. The frameborder attribute specifies whether or not to display a border around a frame.

12.hidden

Example:

The hidden attribute is a boolean attribute. When present, it specifies that an element is not yet, or is no longer, relevant. Browsers should not display elements that have the hidden attribute specified.

13.multiple

Example:

Hold down the Ctrl (windows) / Command (Mac) button to select multiple options.

The multiple attribute is a boolean attribute. When present, it specifies that the user is allowed to enter/select more than one value.

14.muted

Example:

The muted attribute is a boolean attribute. When present, it specifies that the audio output of the video should be muted.

15.readonly

Example:

The readonly attribute is a boolean attribute. When present, it specifies that an input field or textarea is read-only. A read-only field cannot be modified (however, a user can tab to it, highlight it, and copy the text from it).

The noshade attribute is a boolean attribute. When present, it specifies that a horizontal line should render in one solid color (noshaded), instead of a shaded color.

The required attribute is a boolean attribute. When present, it specifies that the element must be filled out before submitting the form.

18.selected

Example:

The selected attribute is a boolean attribute. When present, it specifies that an option should be pre-selected when the page loads. The pre-selected option will be displayed first in the drop-down list.

19.spellcheck

Example:

The spellcheck attribute specifies whether the element is to have its spelling and grammar checked or not.
The following can be spellchecked:

  • Text values in input elements (not password)
  • Text in textarea elements
  • Text in editable elements
